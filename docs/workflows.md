## User Journey Map

### Onboarding

1. Welcome → Setup Org → Setup Stores → Add Initial Items

### Daily Operations

1. Dashboard → Check Alerts → Receive Stock → Issue Items → Review Reports

### Stock Management

1. Items List → Filter/Search → Edit Items → Adjust Stock → View History

### Monthly Processes

1. Run Reports → Stock Count → Reorder Planning → Supplier Orders

### Mobile Work

1. Stock Count → Quick Receiving → Item Lookup → Barcode Scanning

## Key User Flows

### Receive Stock Flow

1. Dashboard → New Transaction → Receive → Select Store → Add Items → Review → Finalize

### Issue Items Flow

1. Items List → Select Item → Quick Issue → Select Store → Quantity → Reason → Confirm

### Low Stock Management

1. Dashboard Alert → Low Stock Report → Create Purchase Order → Receive Stock

### Stock Count Process

1. Mobile App → Select Store → Scan/Count Items → Review Variances → Apply Adjustments
